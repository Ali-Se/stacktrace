Description

Class that on construction copies minimal information about call stack into its internals and provides access to that information.

Template Parameters

  1. typename Allocator

    Allocator to use during stack capture.

  1. explicit basic_stacktrace(std::size_t max_depth = static_cast< std::size_t >(-1), 
                          const allocator_type & a = allocator_type()) noexcept;
    Stores the current function call sequence inside the class.

    Complexity: O(N) where N is call sequence length, O(1) if BOOST_STACKTRACE_USE_NOOP is defined.

    Async-Handler-Safety: Safe if Allocator construction, copying, Allocator::allocate and Allocator::deallocate are async signal safe.

    Parameters:

    max_depth

    max stack depth

    Throws:

    		 Nothing. Note that default construction of allocator may throw, hovewer it is performed outside the constructor and exception in allocator_type() would not result in calling std::terminate.
